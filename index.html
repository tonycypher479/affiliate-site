Yes — you can change your GitHub username, but be aware it has major consequences, especially for GitHub Pages.

✅ How to Change Your GitHub Username

🔹 Step-by-Step:

  1. Go to https://github.com/settings/profile

  2. Under the "Username" section, click "Edit"

  3. Type your new desired username

  4. GitHub will check if it’s available

  5. Click “Change username” if available

⚠️ What Happens After Changing Username?

❗ GitHub Pages URLs Will Change:

If your GitHub Pages site was:

pgsql
https://old-username.github.io/repo-name/

It will become:

pgsql
https://new-username.github.io/repo-name/

Old links to old-username.github.io will stop working.

🔁 Git URLs Will Change:

Your repositories will move to the new username:

  • Old: https://github.com/old-username/repo-name.git

  • New: https://github.com/new-username/repo-name.git

You must update Git remotes on your computer:

bash
git remote set-url origin https://github.com/new-username/repo-name.git

💬 Followers, Stars, Issues, Contributions?

  • These will be retained.

  • GitHub automatically redirects most links (but not GitHub Pages).

🚨 Summary Before You Change:

ItemWhat Happens
GitHub Pages siteURL changes (you must update all links)
Remote URLsMust be updated in local Git configs
Followers, starsKept (GitHub redirects repos)
Custom domainsStill work (but update if username is in URL)
Package registriesMay break; double-check them
